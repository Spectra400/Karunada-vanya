# Karunada-Vanya 🌿🐘

Karunada-Vanya is an Android-based wildlife awareness and realtime movement alert application developed for Karnataka wildlife safety and forest awareness.

The application helps users:
- Explore wildlife information
- Listen to forest and wildlife sounds
- Receive realtime wildlife movement alerts
- Share wildlife sightings with the community

---

# 📱 Features

## 🐾 Wildlife Wiki
Detailed wildlife information including:
- Description
- Habitat
- Diet
- Conservation Status
- Interesting Facts
- Safety Precautions

Animals Included:
- Black Panther
- Leopard
- Elephant
- Tiger
- Cheetah
- Hornbill

---

## 🔊 Forest Sounds
Nature and wildlife audio playback using MediaPlayer.

Included Sounds:
- Bird Chirping
- Elephant Trumpet
- Forest Ambience
- Tiger Roar
- Cheetah Roar

---

## 🚨 Realtime Movement Alerts
Community-driven wildlife alert system where users can:
- Report wildlife sightings
- Add animal name
- Add description
- Add location
- Add district

Alerts are synchronized instantly using Firebase Realtime Database.

---

## 📍 District-Based Filtering
Users select their district on first launch.

Examples:
- Mysore
- Hassan
- Dakshina Kannada

The app shows only relevant alerts for the selected district.

---

## ⏳ Auto Alert Expiration
Old alerts automatically expire after a fixed duration to prevent outdated information.

---

## 🎨 Premium UI
Modern cinematic forest-themed UI with:
- Glassmorphism cards
- Material 3 design
- Dark forest theme
- Rounded cards
- Realtime alert cards

# 🔥 Firebase Integration

Firebase Realtime Database is used for:
- Realtime synchronization
- Wildlife alert storage
- Instant RecyclerView updates

Each alert stores:
- Animal Name
- Description
- Location
- District
- Timestamp

---

# 🧠 Architecture

User → Android App → Firebase Realtime Database → Realtime Sync → RecyclerView UI
