# 🌿 About The Project

Karunada-Vanya is a smart wildlife awareness and realtime movement alert Android application developed for Karnataka forest and wildlife safety awareness.

The application helps users:
- Explore wildlife information
- Learn animal habitats and conservation details
- Listen to immersive forest sounds
- Receive realtime wildlife movement alerts
- Share wildlife sightings within their district

The project focuses on improving:
- Community safety
- Wildlife awareness
- Forest conservation awareness
- Realtime communication during wildlife movement situations

---

# 🚀 Key Features

✅ Realtime Wildlife Alerts using Firebase  
✅ District-Based Alert Filtering  
✅ Wildlife Wiki with Animal Information  
✅ Forest Sounds Module  
✅ Safety Precaution Guidelines  
✅ Modern Forest-Themed UI  
✅ RecyclerView Dynamic Cards  
✅ SharedPreferences-based User Settings  
✅ Auto Expiration of Old Alerts

# 🔥 Firebase Integration

Firebase Realtime Database is used to:
- Store wildlife alerts
- Synchronize alerts in realtime
- Update RecyclerView dynamically
- Share district-based wildlife notifications instantly

Each alert contains:
- Animal Name
- Description
- Location
- District
- Timestamp
